## Getting Started

1. Clone the repository

```bash
git clone <this_repo_url>
```

2. From the root folder, `The-Photo-Diary`, get the required packages

```bash
cd photo_diary
flutter pub get
```

3. Run the Flutter application

```bash
flutter run
```

> Use `--release` for release build

<br>
